# APPFORGE-009 Beta Product Pipeline Manual Validation

1. Hard refresh the deployed or local app.
2. Confirm Product Builder through Approval Ledger still render.
3. Open Beta Product Pipeline MVP in Product Factory Mode.
4. Create a manual pipeline item.
5. Set stage to `product_draft` and priority to `high`.
6. Check Product Builder readiness.
7. Add blockers and next action; confirm blocked/needs-review styling.
8. Save Pipeline and Load Pipeline.
9. Click an explicit import checkpoint button when a source local draft exists.
10. Export JSON and verify safety flags.
11. Copy Markdown and verify stage, priority, readiness score and approval warning.
12. Confirm there is no database, API, payment, checkout creation, webhook, n8n, marketplace publication, social, email, WhatsApp or auto-publication action.
