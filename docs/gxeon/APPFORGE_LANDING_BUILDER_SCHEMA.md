# Landing Builder Schema

`LandingBuilderDraft` stores product idea, niche, audience, problem, offer, promise, base price, delivery format, selected platforms, landing goal, page style, CTA mode, proof notes, approval notes, and timestamps.

`LandingBuilderOutput` contains hero, subheadline, avatarProblem, transformation, offerStack, deliverables, pricingBlock, ctaCopy, faq, proofPlaceholders, legalSafetyNotes, pageSections, tailwindWireframe, humanApprovalChecklist, riskWarnings, and nextSteps.

`LandingBuilderExport` wraps draft, landing output, visible context payload, prompt, markdown, safety flags, and exportedAt.

Safety flags are manualFirst, noGuaranteedIncome, noRealCheckout, noLivePayments, noAutoDeploy, noMarketplaceApiExecution, and localOnlyDraft.
