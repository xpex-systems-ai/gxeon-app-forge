# Checkout Blueprint Manual Validation

1. Open Product Factory Mode.
2. Expand Checkout Blueprint MVP.
3. Optionally click the Product Builder or Marketplace Pack local draft import buttons.
4. Fill product idea, audience, offer or promise, base price, pricing model, support, and guarantee notes.
5. Click **Gerar Checkout Blueprint**.
6. Confirm pricing, plans, order bumps, upsells, downsells, thank-you page, guarantee/refund, support, risks, and approval checklist appear.
7. Confirm no real checkout link, payment status, OAuth, gateway call, marketplace API call, or auto-publish action appears.
8. Use **Enviar para Composer** and verify the textarea is populated but not submitted.
9. Copy prompt/Markdown and export JSON locally.
10. Save, reload, and load the local draft from `gxeon.checkoutBlueprint.draft.v1`.
