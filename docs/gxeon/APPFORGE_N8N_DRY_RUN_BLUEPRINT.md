# n8n Dry-Run Blueprint

The MVP produces n8n workflow draft text only. The workflow is explicitly disabled in the MVP and uses a manual review trigger label, not a live webhook.

## Disabled Draft Flow

1. Manual review trigger only.
2. Review product, marketplace, checkout, landing and content context.
3. Validate terms, taxes, refunds, support, claims and delivery.
4. Approve future credential strategy outside the frontend.
5. Defer all executable automation to a later human-approved API layer.
