# GXEON Agent Command Map

The command map is descriptive data only. Commands include module, selector, risk level, local-only flag, external-action-blocked flag and human-approval requirement.

## Safe local commands
- Generate local drafts.
- Import local drafts between GXEON modules.
- Copy Markdown.
- Export JSON.
- Save local drafts.
- Add local approval, beta pipeline and revenue ledger entries.

## Blocked operations
Publishing, payment, checkout activation, API calls, webhooks, email, WhatsApp, social posting and marketplace deploy are explicitly blocked.
