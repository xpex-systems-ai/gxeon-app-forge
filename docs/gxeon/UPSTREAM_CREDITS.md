# Upstream Credits

GXEON App Forge is based on the open-source **bolt.diy** project. The base project is MIT licensed and remains credited as the underlying open-source technology.

GXEON transforms this foundation into a Digital Product Creation OS by adding a productization layer focused on digital products, marketplaces, checkout blueprints, launch workflows, and commercial operating-system design.

This fork must not claim that the original upstream code was built from scratch by GXEON. Upstream attribution should remain visible in documentation and appropriate product surfaces.
