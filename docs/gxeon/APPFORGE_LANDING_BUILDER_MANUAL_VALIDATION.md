# Landing Builder Manual Validation

1. Open Product Factory Mode and confirm Product Builder, Marketplace Pack, and Checkout Blueprint remain accessible.
2. Open Landing Builder MVP.
3. Import local drafts only by explicit button clicks when available.
4. Fill product, audience, promise, base price, CTA mode, and landing goal.
5. Click **Gerar Landing Blueprint** and confirm a local preview appears.
6. Confirm there is no live deployment status, checkout URL, payment link, gateway action, marketplace API action, or publish action.
7. Click **Enviar para Composer** and confirm the textarea is populated and focused but not submitted.
8. Copy Prompt, Copy Markdown, and Export JSON.
9. Inspect JSON for safety flags and absence of secret-like fields.
